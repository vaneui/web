## npm

```bash
npm install @vaneui/ui
```

## yarn

```bash
yarn add @vaneui/ui
```

## pnpm

```bash
pnpm add @vaneui/ui
```

## Getting Started

After installation, you can start using VaneUI components in your React application:

```tsx
import { Button, ThemeProvider } from '@vaneui/ui';

function App() {
  return (
    <ThemeProvider>
      <Button>Click me</Button>
    </ThemeProvider>
  );
}
```
